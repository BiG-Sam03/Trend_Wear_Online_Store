
# Trend Wear Online Store

This is a simple Python-based online store application that supports user login and registration using SQLite as the database.

## Features

- User registration
- User login
- Database connectivity

## Requirements

- Python 3.x
- MySQL (for future expansion)

## How to run the project

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Trend_Wear_Online_Store.git
   ```

2. Install required packages:
   ```bash
   pip install mysql-connector
   ```

3. Run the application:
   ```bash
   python login.py
   python register.py
   ```
